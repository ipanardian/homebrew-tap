# Homebrew Tap for Ipan Ardian

Official Homebrew tap for Ipan Ardian's projects.
 
## Installation
 
```bash
brew tap ipanardian/tap
```

## Available Formulae
### lu-hut
A powerful modern alternative to the Unix ls command with beautiful box-drawn tables, tree view, git integration, and more.

```bash
brew install lu-hut
``
### Features:
- Beautiful box-drawn tables with colored borders
- Stunning tree view display
- Real-time Git status monitoring
- Time-aware color grading
- Advanced filtering capabilities
- And much more!
For more information: https://github.com/ipanardian/lu-hut
